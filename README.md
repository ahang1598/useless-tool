# useless-tool

A collection of fun and useless tools. 一堆"没有用但有点上头"的小工具，主打一个认真做没用的事。

> 认真的，没用。Seriously, useless.

## 在线工具 / Tools

- **时光机 / Time Machine** — 把等待包装成穿越。设定时长，真的让你等那么久，然后宣布你穿越成功。
- **自律神器 / Discipline Timer** — 学习时计时以 **100 倍速**膨胀，让你瞬间成就感爆棚；休息时则按真实 1× 速度计时，残忍诚实。底部展示虚拟学习、真实学习、累计休息三项时长，以及由"虚拟学习 ÷ (虚拟学习+休息)"精心美化出来的「自律指数」。
- 更多无用工具陆续上线中……

## 技术栈 / Stack

- React 18 + Vite
- Tailwind CSS（自定义 acid / frost / plasma 配色）
- Framer Motion（入场、转场、计时动效）
- React Router
- 中英双语 i18n（localStorage 持久化）
- 数据持久化：自律神器累计时长通过 localStorage 自动存档

## 本地开发 / Develop

```bash
npm install
npm run dev      # 启动开发服务器
npm run build    # 生产构建
npm run preview  # 预览构建产物
```
